Syarat dan Ketentuan

Selamat datang di Data Elemental. Silakan baca Syarat dan Ketentuan ini dengan cermat sebelum menggunakan situs web dan layanan kami. Dengan mengakses atau menggunakan situs web kami, Anda setuju untuk terikat oleh Syarat dan Ketentuan ini. Jika Anda tidak setuju dengan bagian manapun dari syarat-syarat ini, Anda tidak diperbolehkan mengakses situs web ini.

1. Penggunaan Situs Web

Situs web dan isinya ditujukan hanya untuk tujuan informasi. Anda diperbolehkan menggunakan situs web kami untuk tujuan yang sah dan sesuai dengan Syarat dan Ketentuan ini. Anda setuju untuk tidak menggunakan situs web kami dengan cara apa pun yang melanggar hukum atau peraturan yang berlaku, melanggar hak orang lain, atau mengganggu operasi situs web kami.

2. Hak Kekayaan Intelektual

Konten, desain, grafik, dan materi lainnya di situs web kami dilindungi oleh hak cipta dan undang-undang kekayaan intelektual lainnya. Anda tidak diperbolehkan untuk menggandakan, memodifikasi, mendistribusikan, atau menggunakan kekayaan intelektual kami tanpa persetujuan tertulis sebelumnya. Logo Data Elemental dan semua merek dagang terkait dimiliki oleh Data Elemental dan tidak boleh digunakan tanpa izin.

3. Penafian Jaminan

Situs web dan isinya disediakan dalam kondisi "apa adanya" dan "tersedia sebagaimana adanya" tanpa jaminan apapun, baik tersurat maupun tersirat. Kami tidak menjamin bahwa situs web kami akan selalu tersedia, bebas dari kesalahan, atau bebas dari virus atau komponen berbahaya lainnya. Kami tidak memberikan pernyataan atau jaminan mengenai keakuratan, kelengkapan, atau keandalan konten apa pun di situs web kami.

4. Batasan Tanggung Jawab

Sejauh diizinkan oleh hukum yang berlaku, Data Elemental tidak bertanggung jawab atas segala kerusakan langsung, tidak langsung, insidental, khusus, konsekuensial, atau hukuman yang timbul akibat atau terkait dengan penggunaan situs web atau isinya. Ini termasuk, namun tidak terbatas pada, kerusakan atas kehilangan keuntungan, data, goodwill, atau kerugian tidak berwujud lainnya.

5. Tautan ke Situs Web Pihak Ketiga

Situs web kami mungkin mengandung tautan ke situs web pihak ketiga yang tidak dimiliki atau dikendalikan oleh Data Elemental. Kami tidak bertanggung jawab atas konten, kebijakan privasi, atau praktik situs web pihak ketiga manapun. Dengan mengakses situs web pihak ketiga melalui tautan di situs web kami, Anda melakukannya atas risiko Anda sendiri.

6. Hukum yang Mengatur

Syarat dan Ketentuan ini akan diatur oleh dan ditafsirkan sesuai dengan hukum {Negara/Negara Bagian Anda}, tanpa memperhatikan ketentuan konflik hukumnya. Setiap perselisihan yang timbul berdasarkan Syarat dan Ketentuan ini akan menjadi kewenangan eksklusif pengadilan yang terletak di {Kota/Negara Bagian Anda}.

7. Perubahan pada Syarat dan Ketentuan Ini

Data Elemental berhak untuk memodifikasi atau merevisi Syarat dan Ketentuan ini kapan saja tanpa pemberitahuan sebelumnya. Penggunaan situs web kami secara terus-menerus setelah perubahan tersebut merupakan penerimaan Anda terhadap Syarat dan Ketentuan yang telah direvisi. Kami menyarankan Anda untuk meninjau Syarat dan Ketentuan ini secara berkala untuk setiap pembaruan.
